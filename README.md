# phpAPI_MongoDB

-----------

Connection DataBase:

    ..pathDB/mongod
    
Connection API:

    localhost/index.php

Add new document:

    localhost/phpmongodb/insert.php
    
Show collections:

    localhost/input2.php
    
or in terminal

    mongo
    use "DBname"
    show collections
    
Show documents:

    localhost/view.php?id=""
    
or in terminal

    mongo
    use "DBname"
    db."CollectionName".find().pretty()
